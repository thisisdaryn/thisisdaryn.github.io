More questions:

  • What day of the year had the most flights scheduled?

  • What day of the year had the least flights scheduled?

Getting the day with the most flights scheduled

Steps:

  • group by month and day

  • use summarise to get the counts for each day

  • sort in descending order

flight_counts <- group_by(df, month, day) %>% 
  summarise(Scheduled = n()) %>%  
  arrange(desc(Scheduled))

Getting the day with the least flights scheduled

Steps:

  • group by month and day

  • use summarise to get the counts for each day

  • sort in (ascending: default) order

flight_counts2 <- group_by(df, month, day) %>% 
  summarise(Scheduled = n()) %>% 
  arrange(Scheduled)

Making a plot of number of flights by day

Steps:

  • first create a date variable from the year, month and day

  • group the data frame by the date, then take the count for each day

  • plot the counts for each day for the year

daily_flights <- mutate(df, date = ISOdate(year, month, day)) %>% group_by(date) %>% summarise(Scheduled = n())

plot(Scheduled~date, data = daily_flights,
     type = "l")
